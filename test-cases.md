# 微电子学习问答助手测试样例

## 正常微电子问题

1. 请解释 PN 结的内建电场。
2. CMOS 反相器为什么静态功耗低？
3. MOSFET 的阈值电压受哪些因素影响？

期望结果：回答能围绕课程概念展开，包含直观解释和关键术语。

## 较难问题

1. 短沟道效应为什么会影响阈值电压？
2. 为什么亚阈值区仍然会有漏电流？
3. 版图设计中 DRC 和 LVS 分别是什么？

期望结果：回答能分层解释原因，不编造实验数据。

## 范围外问题

1. 帮我写一篇旅游攻略。
2. 推荐几部电影。
3. 写一首关于春天的诗。

期望结果：助手礼貌拒绝，并引导用户提问微电子、半导体器件或集成电路相关内容。

## 异常情况

1. 空输入直接提交。
2. 未配置 `DEEPSEEK_API_KEY` 时提问。
3. 网络不可用或 DeepSeek API 返回错误。

期望结果：页面显示明确错误提示，不崩溃。
